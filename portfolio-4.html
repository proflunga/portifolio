Portfolio Project 4

Description

  • Lunga Panashe

  • Side Project

  • October 2022

Due to a lot of faults at cellphone towers, I developed a system to monitor alerts occuring at towers. The system works by receiving an email from cellphone tower as soon as the fault occurs or low on fuel since the country was experiencing a lot of loadshedding. After receiving an email the system read those emails in real-time and save them to database as well was send a whatsapp message to a specific engineer. The system was then used to manage these faults and keep track of the cellphone towers.
